FragView

Terms of Use

FragView is an open-source application for metabolite fragmentation, it heavily relies on the code base from MolView (see http://molview.org/). The website will continue to be updated to add new features and address any issues discovered.

Copyright

FragView is an open-source project that possesses copyright protection from its GNU APGL license. If you have any copyright questions about the project, please let us know through GitHub.

Additional Site Information

FragView will continuously be updated for improvement, which means the site can change appearance without any notice to the users until the update is released. If you wish to work with an older version of the site, please see our past pushes to the GitHub branch at https://github.com/mgasoit/FragView. Furthermore, if you want faster load times, it is recommended to download the application from the Master GitHub branch and follow the instructions to run the application on a localhost server.

Reliability of Data

FragView uses RDKit Python and RDKitJS to deliver dynamic data for molecules and their fragments. For more information on RDKit, see https://www.rdkit.org/docs/GettingStartedInPython.html.

All third-party data is not owned by FragView and protected by their own, respective copyrights. If you possess the copyright for any data involved in this application, please post an issue at the GitHub branch (see https://github.com/mgasoit/FragView) so we can evaluate your claim and take the necessary actions to remove the data or properly identify its owner.

Privacy Policy

No data that you own will be stored in a server or database.
